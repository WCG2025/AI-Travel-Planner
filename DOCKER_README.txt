📦 文件说明
--------------------------------------------------------------------------------
- ai-travel-planner-docker-image.tar    Docker 镜像文件（67 MB）
- docker-compose.yml                     Docker Compose 配置文件
- .env.example                           环境变量模板
- BUILD_AND_RUN.md                       详细运行指南 ⭐⭐⭐⭐⭐


🚀 快速开始
--------------------------------------------------------------------------------

1. 导入镜像
   docker load -i ai-travel-planner-docker-image.tar

2. 创建 .env 文件（复制 .env.example 并填写 API Keys）
   DEEPSEEK_API_KEY=你的_deepseek_key
   AMAP_WEB_SERVICE_KEY=你的_amap_web_service_key

3. 启动应用
   docker-compose up -d

4. 访问应用
   http://localhost:3000

⚠️ 重要提示
--------------------------------------------------------------------------------
- 镜像已内置大部分环境变量（Supabase、科大讯飞、高德地图）
- 只需提供 2 个服务端 API Key：
  * DEEPSEEK_API_KEY
  * AMAP_WEB_SERVICE_KEY
- 所有 API 都有免费套餐，申请方法见文档

🐛 故障排查
--------------------------------------------------------------------------------
容器无法启动：
  → 检查端口 3000 是否被占用
  → 查看日志：docker logs ai-travel-planner

页面打不开：
  → 确认容器运行：docker ps
  → 检查防火墙设置

功能报错：
  → 检查浏览器控制台
  → 确认 API Keys 已配置
  → 查看容器日志
