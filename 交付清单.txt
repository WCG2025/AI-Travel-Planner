五、快速开始（评审老师）
================================================================================

步骤 1：导入镜像
--------
docker load -i ai-travel-planner-docker-image.tar

步骤 2：创建 .env 文件
--------
DEEPSEEK_API_KEY=your_deepseek_key
AMAP_WEB_SERVICE_KEY=your_amap_web_service_key

步骤 3：启动应用
--------
方法A（推荐）：
docker-compose up -d

方法B：
docker run -d --name ai-travel-planner -p 3000:3000 \
  -e DEEPSEEK_API_KEY="your_key" \
  -e AMAP_WEB_SERVICE_KEY="your_key" \
  ai-travel-planner:latest

步骤 4：访问应用
--------
浏览器打开：http://localhost:3000

八、故障排查
================================================================================

问题1：容器无法启动
解决：
- 检查端口 3000 是否被占用
- 查看日志：docker logs ai-travel-planner
- 确认环境变量已提供

问题2：页面打不开
解决：
- 确认容器运行：docker ps
- 检查端口映射：docker port ai-travel-planner
- 检查防火墙设置

问题3：某个功能报错
解决：
- 检查浏览器控制台（F12）
- 确认对应的 API Key 已配置
- 查看容器日志

功能对应的 API：
- AI 生成 → DEEPSEEK_API_KEY
- 语音识别 → 科大讯飞配置（已内置）
- 地图显示 → 高德地图配置（已内置）
- 地理编码 → AMAP_WEB_SERVICE_KEY
- 数据存储 → Supabase 配置（已内置）
